# Encrypted Substitution Cipher Tool

## Overview
This project implements an encryption and decryption system using frequency analysis techniques in C++. The program processes large text inputs to encode and decode messages based on substitution cipher logic.

The system reads plaintext from files, performs encryption or decryption operations, and validates outputs using a dictionary-based approach.

## Features
- Encrypts plaintext using substitution cipher logic  
- Decrypts encrypted text using frequency analysis  
- Handles large file inputs  
- Uses dictionary validation for decrypted output  

## How to Run
Compile using:
g++ main.cpp -o cipher
./cipher
